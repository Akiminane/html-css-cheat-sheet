html css cheat table

Html

Corps de page

Tags Nom Descritpion
<html> html Élément racine du document HTML. Il englobe toute la page et indique au navigateur qu’il s’agit d’un document HTML.
<head> head Contient les informations sur la page, utilisées par le navigateur, non visibles par l’utilisateur.
<body> body Contient tout le contenu visible de la page web.
<link> Link Permet de lier une ressource externe au document HTML

Mise en page

Tags Nom Descritpion
<h1> -> <h6> Titre Titres hiérarchiques qui définissent la structure du contenu.
<div> Div Conteneur sans sens sémantique, utilisé pour regrouper des éléments.
<p> Paragraphe Représente un paragraphe de texte.
<strong> Strong Met en valeur un texte avec une importance forte.
<em> Emphasized Met en valeur un texte avec une emphase (insistance).
<br> Break Force un retour à la ligne dans le texte.
<span> span Conteneur inline neutre, sans signification sémantique.

Semantique

Tag Nom Description
<header> header Contient le contenu introductif d’une page ou d’une section.
<nav> Barre de navigation Contient un ensemble de liens de navigation.
<main> main Contient le contenu principal unique de la page.
<section> section Regroupe un contenu thématique.
<article> article Contenu autonome et réutilisable.
<aside> aside Contenu secondaire ou complémentaire.
<footer> pied de page Contient les informations de fin d’une page ou d’une section.
<figure> figure Contient un contenu illustratif autonome.
<figcaption> figcaption Fournit une légende pour un <figure>.

media

tag Nom Description
<img> Image Affiche une image dans la page.
<video> video Intègre une vidéo dans la page.
<audio> Audio Intègre un fichier audio.
<source> Source Définit une source média pour <video> ou <audio>.
<embed> embed Intègre une ressource externe (PDF, média).

CSS

Typographie

Propriété Description
font-family Définit la police de caractères utilisée.
font-size Définit la taille du texte.
font-weight Définit l’épaisseur du texte.
font-style Définit le style du texte.
line-height Définit l’espace entre les lignes.
letter-spacing Définit l’espace entre les lettres.
word-spacing Définit l’espace entre les mots.
text-align Définit l’alignement horizontal du texte.
text-transform Modifie la casse du texte.
text-decoration Ajoute une décoration au texte.
text-overflow Gère le débordement du texte.

Layout

Propriete Description
display Définit le type de boîte d’un élément et son comportement dans le layout.
position Définit le mode de positionnement d’un élément.
float Place un élément à gauche ou à droite.
clear Empêche un élément de se placer à côté d’un élément flottant.
width Définit la largeur d’un élément.
height Définit la hauteur d’un élément.
max-width Limite la largeur maximale.
min-width Définit la largeur minimale.
box-sizing Définit comment la taille totale est calculée.
margin Définit l’espace extérieur d’un élément.
padding Définit l’espace intérieur d’un élément.
overflow Gère le débordement du contenu.
z-index Définit l’ordre de superposition.
visibility Contrôle la visibilité sans retirer l’espace.
